## Image Guidelines

- **Format**: JPG or PNG
- **Dimensions**: 1920x1080px (16:9 aspect ratio)
- **Quality**: High resolution, optimized for web
- **Content**: Professional, modern, relevant to the service
- **Colors**: Should work well with brand colors and white text overlay
- **Lighting**: Ensure good contrast for text readability

## Brand Colors

- Primary: #303661 (Dark Blue)
- Secondary: #e0ae00 (Gold)
- Accent: #8f7622 (Bronze)

The hero slider includes a dark overlay (bg-black/50) to ensure text readability over the background images.

## Performance Optimizations

- **Preloading**: All hero images are preloaded in the HTML head for instant display
- **Progressive Loading**: Loading indicator shows while images are being preloaded
- **Caching**: .htaccess file configured for 1-year browser caching
- **Compression**: Images are served with compression enabled
- **Hardware Acceleration**: CSS transforms use `will-change-transform` for GPU acceleration
- **WebP Format**: Optimized WebP format for smaller file sizes and faster loading
